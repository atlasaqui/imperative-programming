# Listas de Exercícios

Este diretório contém as listas de exercícios propostas durante a disciplina de Programação Imperativa.

## Organização

Cada lista de exercícios será organizada em um subdiretório próprio, contendo:
- Enunciado dos exercícios
- Soluções implementadas em C
- Comentários e observações relevantes

*As listas serão adicionadas conforme forem sendo disponibilizadas.*
